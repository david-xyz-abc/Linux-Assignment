Ifconfig Command Overview

The ifconfig command in Linux is used to configure, manage, and display network interfaces:

  • View detailed information about network interfaces.
  • Enable or disable network interfaces.
  • Assign IP addresses and netmasks to interfaces.

Basic Usage

Common Commands:

  • ifconfig: Displays information about all active interfaces.
  • ifconfig [interface]: Shows details for a specific interface.
  • ifconfig [interface] up: Activates the specified interface.
  • ifconfig [interface] down: Deactivates the specified interface.

IP Address Configuration

The ifconfig command allows manual configuration of IP addresses:

  • ifconfig [interface] [IP address]: Assigns an IP address to the interface.
  • ifconfig [interface] netmask [subnet mask]: Sets the subnet mask.
  • ifconfig [interface] [IP address] netmask [subnet mask]: Assigns both IP address and subnet mask.

Advanced Features

Leverage the ifconfig command for advanced network management:

  • ifconfig [interface] mtu [value]: Sets the Maximum Transmission Unit (MTU).
  • ifconfig [interface] promisc: Enables promiscuous mode for packet sniffing.
  • ifconfig [interface] broadcast [address]: Configures the broadcast address.
  • Combine with arp and other tools for comprehensive network management.
